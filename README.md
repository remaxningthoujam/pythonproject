# PYTHON SIMPLE CALCULATOR
This is a basic calculator made using Python. It can perform:
--> Addition(+)
--> Subtraction(-)
--> Multiplication(*)
--> Division(/)
# HOW TO RUN
Install Python
Download the file
Run in terminal:
python calculator.py

# EXAMPLE
Enter fisrt number: 10
Enter Second number: 5
Enter Operator(+, -, *, /): +
Output: 15
