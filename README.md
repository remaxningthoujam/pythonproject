# PYTHON SIMPLE CALCULATOR
### This is a basic calculator made using Python. It can perform:
- Addition(+)
- Subtraction(-)
- Multiplication(*)
- Division(/)
## HOW TO RUN
- Install Python
- Download the file
- Run in terminal:
- python calculator.py

## EXAMPLE
- Enter first number: 10
- Enter second number: 5
- Enter operator (+, -, *, /):+
- Output: 15
